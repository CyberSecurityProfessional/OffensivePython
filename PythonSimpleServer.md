# Python simple http server
```
Python2:
python -m SimpleHTTPServer --> default port will be 8000 --> http://127.0.0.1:8000
python -m SimpleHTTPServer 80
Python3:
python3 -m http.server 80
python3 -m http.server 80
python -m http.server --bind 0.0.0.0 80
```
